A vector of tag values of a single type.

Each element in the vector represents the value of tag for the chunk with the corresponding index.

Pushes a new tag onto the end of the vector.

Safety

Ensure the tag pointed to by ptr is representative of the tag types stored in the vec.

ptr must not point to a location already within the vector.

The value at ptr is copied into the tag vector. If the value is not Copy, then the caller must ensure that the original value is forgotten with mem::forget such that the finalizer is not called twice.
